# BookReviewHub
An application where users can: 
  1. Browse books
  2. View book details, and reviews
  3. Submit reviews for books
Each book can have many reviews. Each user can create reviews and only manage their own reviews.

 b. author

 d. genre

 f. cover_url

  c. rating
